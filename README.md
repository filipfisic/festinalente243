# FESTINA LENTE 243 - Moderna Web Stranica

Moderna, responzivna one-page web stranica za FESTINA LENTE 243 - putničku agenciju.

## Struktura Projekta

```
FORKO/
├── index.html          # Glavna HTML stranica (one-pager)
├── css/
│   ├── reset.css       # CSS reset
│   ├── style.css       # Glavni stilovi
│   └── responsive.css  # Responsive stilovi
├── js/
│   └── main.js         # JavaScript funkcionalnosti
└── images/             # Slike i resursi
```

## Funkcionalnosti

- ✅ **One-page design** - Sve sekcije na jednoj stranici
- ✅ **Responzivni dizajn** - Optimizirano za sve uređaje
- ✅ **Smooth scrolling** - Glatko skrolanje između sekcija
- ✅ **Animacije** - Fade-in animacije na scroll
- ✅ **Testimonials slider** - Automatski slider za recenzije
- ✅ **Mobile menu** - Hamburger menu za mobilne uređaje
- ✅ **Kontakt sekcija** - Kontakt informacije s kartičnim dizajnom
- ✅ **Back to top button** - Gumb za povratak na vrh
- ✅ **Galerija auto-rotacija** - Automatska rotacija galerije (6 slika odjednom)
- ✅ **Dark mode dizajn** - Tamniji dizajn s plavim i narančastim akcentima
- ✅ **Animacije** - Scroll animacije, hover efekti, fade-in animacije

## Sekcije

1. **Hero** - Glavna sekcija s naslovom i pozivom na akciju
2. **O meni** - Informacije o Ivanu Forku
3. **Iduća putovanja** - Prikaz nadolazećih putovanja
4. **Galerija** - Slike s putovanja
5. **Recenzije** - Testimonials slider
6. **Kontakt** - Kontakt informacije i forma

## Ikone

Koristi Font Awesome 6.4.0 za ikone.

## Responsive Breakpoints

- **Desktop:** > 1200px
- **Tablet:** 768px - 1199px
- **Mobile:** < 768px

## Kako koristiti

1. Otvorite `index.html` u web pregledniku
2. Za produkciju, postavite fajlove na web server
3. Ažurirajte kontakt informacije u HTML-u
4. Dodajte stvarne linkove društvenih mreža
